## DML

Modify data in tables.

| Topic | Description |
|-------|-------------|
| [INSERT](insert.md) | Add rows to a table |
| [UPDATE](update.md) | Modify existing rows |
| [DELETE](delete.md) | Remove rows |
| [COPY INTO](copy-into-table.md) | Bulk load from files or stage |
| [MERGE](merge.md) | Conditional insert, update, or delete |
